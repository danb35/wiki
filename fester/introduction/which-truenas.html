iXSystems has two current and distinct TrueNAS products each of which has several supported versions:

  • TrueNAS CORE
    • Current version: 13.3
  • TrueNAS SCALE
    • Current version: Dragonfish 24.04
    • Previous version: Cobia 23.10
    • Pre-release version: Electric Eel 24.10

TrueNAS CORE is built on top of FreeBSD and grew from FreeNAS, which dates back to about 2011. In addition to developing TrueNAS, iXSystems has contributed heavily to the development of FreeBSD and OpenZFS. Even Gomez, who has been around for a couple of centuries, and who has seen presidents and even countries come and go, is impressed with this pedigree.

TrueNAS SCALE is essentially the same functionality but migrated to Debian Linux and saw its first release in early 2022. The change of technology base and ongoing development has resulted in some significant divergence between the two products at a very detailed level, but at a high level they have pretty similar functionality.  

Bottom Line (TL;DR)

In general terms we currently recommend (as does iXSystems) that you install TrueNAS SCALE rather than CORE: 

  • TrueNAS SCALE has been the major focus of iXSystems' development efforts over the last four or more years, and that seems unlikely to change. Almost all further improvements in functionality are going to be delivered in TrueNAS SCALE rather than CORE.
  • TrueNAS SCALE has had four major versions and a multitude of minor versions, and can be considered both stable, fully production quality and proven in the field.
  • TrueNAS CORE 13.3 has now been released, and it's likely this will be the last release of CORE barring security fixes.  Unless you have a philosophical or technical objection to Linux or preference for FreeBSD, you will want to install TrueNAS SCALE. If storage is your only need, CORE may still be a slightly better choice. If these do not apply or If you want to run applications or virtual machines, or position yourself for long-term upgrades, SCALE is almost certainly the best choice:

If you choose SCALE, then we recommend that you install the Dragonfish 24.04 version of TrueNAS SCALE rather than Cobia 23.10: 

  • TrueNAS SCALE Bluefin 22.12 is now end-of-life and no longer supported by iXSystems.
  • TrueNAS SCALE Cobia 23.10 is still supported, but will soon be end-of-life and no longer supported by iXSystems.
  • TrueNAS SCALE Dragonfish 24.04 has had several minor releases to address bugs found after first release, is now fully recommended by iXsystems and can be considered full stable and will have support for a reasonable time.

If you're reading this at a later date, our general recommendation is that you install the current release, once it's had a couple of point releases.  Thus, if the current version is 24.10.0, you may want to install the latest release of 24.04 instead.  But if the current version is 24.10.2, the major bugs have likely been worked out and it's probably safe to install.

Common Features

Both SCALE and CORE support:

  • ZFS for managing storage
  • File-sharing protocols of SMB and NFS, and block-storage protocol of iSCSI.
  • Multiple users (with attendant security permissions), including integrating with LDAP or Active Directory environments.
  • Running VMs and / or containerised apps

Other alternatives

In the interests of fairness - and because we are enthusiastic users rather than iXsystems employees - we would like to mention alternatives to TrueNAS and indicate (without being experts in these alternatives) how they differ from TrueNAS:

  • unRAID - Supports BTRFS, another file system similar to but different from ZFS. Supports mixed sized disks.  Paid software only; there's no free version of unRAID.  Loses points for using a .net TLD when they aren't an ISP.
  • Open Media Vault - Free, Linux-based NAS software.  Can be made to support ZFS, but it doesn't appear to be well-integrated.
  • Hex OS - actually built on top of TrueNAS SCALE, this (as yet unreleased software) is intended to provide a simplified UI for both set-up and ongoing management in home / small business environments.
  • XigmaNAS - Continues the original FreeNAS code base, FreeBSD-based.
